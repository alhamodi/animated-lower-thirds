# ☽✦ Animated Lower Thirds — الإضافة العربية المطوّرة لـ OBS

شريط سفلي (Lower Thirds) احترافي للبث المباشر والمونتاج مصمم خصيصاً باللغة العربية. يتضمن 26 تصميماً مميزاً، لوحة تحكم فورية، إمكانية السحب والإفلات للتحكم الدقيق بالموضع، ومكتبة خطوط مدمجة محلياً بالكامل لتشغيل الإضافة بدون إنترنت.

---

## ✨ المميزات الجديدة والمطوّرة

* **26 قالب جاهز:** تصاميم مستوحاة من الهوية العربية والإسلامية (مثل الزمرد، والذهبي الملكي، وتصاميم رمضان والأعياد، والشريط الشامل، والموزاييك التراثي النيوني).
* **ترتيب هرمي محسّن للنصوص:** تم ضبط الهيكلية بحيث يُعرض **عنوان الدرس أو الكلمة** في السطر العلوي الأبرز (بخط عريض كبير)، ويُعرض **اسم المحاضر أو الشيخ** في السطر السفلي الفرعي لترتيب بصري أكثر وضوحاً وجاذبية.
* **مكتبة خطوط عربية مدمجة (Local Fonts):** تم دمج خطوط فخمة واحترافية مباشرة في المشروع بصيغة `.woff2` عالية الأداء لتعمل دون الحاجة لاتصال بالإنترنت، ومنها:
  * عائلة خطوط **ثمانية (Thmanyah)** الأنيقة كخط افتراضي للمشروع.
  * الخط الفخم **IBM Plex Sans Arabic**.
  * خط المصحف الشريف (**KFGQPC Uthman Taha Naskh**).
  * خطوط الهوية الرسمية وثقافة الأعوام (عام الإبل، وعام الحرف اليدوية).
* **التنقل الحر (Drag & Drop):** اسحب الشريط بالماوس من شاشة المعاينة في لوحة التحكم لوضعه في أي مكان بدقة! يتم تحويل الأبعاد والمسافات تلقائياً إلى نِسب مئوية (`%`) لتناسب جميع دقات البث (1080p, 720p) عبر تقنية `requestAnimationFrame`.
* **لوحة تحكم ذكية ومتجاوبة:** واجهة تحكم حديثة تدعم (الهواتف، الأجهزة اللوحية، الكمبيوتر) للتحكم بالنص والألوان بحرية كاملة وعن بُعد.
* **ثبات ومزامنة فورية (Robust Anti-Freeze):** استخدام تقنية `BroadcastChannel API` مع معالجة ذكية للـ API Polling تمنع تعارض الأوامر وتجمّد النصوص أثناء البث.
* **دعم ديناميكي للمسارات (No CORS Issues):** يكتشف المشروع أصل الرابط تلقائياً (`window.location.origin`) لطلب البيانات، مما يقضي تماماً على مشاكل حظر المتصفح (CORS preflight).

## 🚀 أسرع طريقة للاستخدام (في أقل من دقيقة!)

### 1. حمّل الإضافة

* **بالطريقة العادية:** اضغط على الزر الأخضر فوق (`Code > Download ZIP`) وفك الضغط عن الملف.
* **أو للمطورين عبر الـ Terminal:**

```bash
git clone https://github.com/alhamodi/animated-lower-thirds.git
cd animated-lower-thirds
```

### 2. افتح لوحة التحكم

انقر نقراً مزدوجاً على ملف `index.html` ليفتح في متصفحك (يفضل Chrome). ستجد خط "ثمانية" يزين الواجهة والقوالب افتراضياً.

### 3. اِربطها ببرنامج OBS

1. داخل الـ OBS، أضف مصدر متصفح جديد **(Browser Source)**.
2. ضع علامة صح (✓) بجوار **Local File**.
3. انقر على **Browse** واختر ملف `obs.html` (أو القالب المطلوب مباشرة من مجلد `templates`).
4. اجعل العرض (Width) **1920** والارتفاع (Height) **1080**.
5. اضغط **OK**.

> 💡 **ملاحظة لنظام الماك (macOS Fallback):** إذا لم تتحدث البيانات في OBS عند استخدام خيار (Local File)، سيتكفل الكود تلقائياً بتمرير النصوص والتغييرات عبر الـ URL بدقة وسلاسة، أو يمكنك تشغيل سيرفر محلي (مثل `Live Server` في VS Code أو `server.py` المرفق) واستخدام الرابط بدلاً من اختيار الملف.

## 🧪 جودة واستقرار الكود (Testing)

المشروع مغطى بالكامل باختبارات برمجية مؤتمتة لضمان أعلى كفاءة واستقرار أثناء البث:

* **اختبارات الوحدة (Unit Tests):** تم إعداد 10 اختبارات عبر `Vitest` للتحقق من سلامة بناء الكود وحقن النصوص والخطوط بأمان باستخدام `style.setProperty(..., '!important')`.
* **اختبارات النهاية للنهاية (E2E Tests):** تم إعداد 7 اختبارات متكاملة عبر `Playwright` للتأكد من سلاسة نقل البيانات وعزل قنوات الاتصال المتعددة بدون أي تعارض (Race Conditions).

لتشغيل الاختبارات محلياً:

```bash
npm install     # لتثبيت الاعتماديات والاختبارات
npm test        # لتشغيل اختبارات Vitest
npm run test:e2e # لتشغيل اختبارات Playwright المتكاملة
```
